# Simulation results

This directory contains processed output from the simulation study.

## Files

* `simulation_summary_tables.RData`: R workspace containing the data-frame objects used to produce the simulation tables and figures reported in the manuscript.

The file includes tabular summaries only. It does not contain source functions, fitted model objects, temporary workspace objects, or raw replicate-level simulation outputs.

The complete simulation workflow is provided in:

```text
../../R/01_simulation_study.R
```

Raw simulation outputs are not included because they can be computationally large and can be regenerated by running the simulation code.
